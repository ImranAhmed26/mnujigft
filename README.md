# Frontend Project

## Tech

This Application uses the following Technologies

- React
- Next JS 12
- Tailwind CSS
- Axios


## Installation

Install the dependencies and devDependencies and start the server.

```sh
cd to Root Folder
npm i
npm run dev
```
